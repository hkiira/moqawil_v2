## Color Scheme

### Primary Colors

```
Primary:    ███ #667eea (Purple Blue)     - Main CTAs, highlights
Success:    ███ #1BC5BD (Teal)           - Positive metrics, revenue
Warning:    ███ #FFA800 (Orange)         - Caution, pending status
Danger:     ███ #F64E60 (Red)            - Critical, cancelled
Info:       ███ #8950FC (Purple)         - Informational
Secondary:  ███ #e9ecef (Light Gray)     - Neutral elements
```

### Ranking Badges

```
🥇 Gold   - Rank #1 (Gradient: #FFD700 → #FFA500)
🥈 Silver - Rank #2 (Gradient: #C0C0C0 → #808080)
🥉 Bronze - Rank #3 (Gradient: #CD7F32 → #8B4513)
   Gray   - Rank #4+ (#e9ecef)
```

### Status Badge Colors

```
✅ Completed  → Success (Teal #1BC5BD)
⏳ Pending    → Warning (Orange #FFA800)
🚚 Shipped    → Info (Purple #8950FC)
⚙️  Processing → Primary (Blue #667eea)
❌ Cancelled  → Danger (Red #F64E60)
```

## Card Components

### Stat Card Variants

```
┌─────────────────────────────────────┐
│ 📋 Card Title                   [📊] │  ← Icon (float right)
├─────────────────────────────────────┤

│ 2,456                               │  ← Large number (primary color)
│ Description of the metric           │  ← Small label (gray)
│                                     │
└─────────────────────────────────────┘
```

**Available Types**:
- `primary` - Purple blue, main metrics
- `success` - Teal, positive metrics
- `warning` - Orange, caution metrics
- `danger` - Red, critical metrics
- `info` - Purple, informational metrics

## Interactive Elements

### Hover Effects

**Stat Cards**:
- Smooth elevation (translateY -5px)
- Shadow increase
- Color intensifies slightly

**Rows in Best Sellers/Products**:
- Light background on hover
- Text becomes bold

**Buttons**:
- Color inversion on hover
- Slight scale increase

### Loading States

When data is loading:
```
⏳ Chargement des données...
```

When no data available:
```
❌ Aucune donnée disponible
```

## Typography

### Font Sizes

```
H1 (Dashboard Title):     28px, Bold (700)
H5 (Card Titles):         18px, Bold (600)
Card Values:              32px, Bold (700)
Labels:                   14px, Regular (400)
Small Text:               12px, Regular (400)
Badges:                   11px, Bold (600)
```

### Font Family

Primary: System fonts stack (Segoe UI, Roboto, Arial, sans-serif)

### Font Colors

```
Primary Text:      #1f2937 (Dark gray)
Secondary Text:    #6c757d (Medium gray)
Tertiary Text:     #9ca3af (Light gray)
Value Text:        Color-coded by type (#667eea, #1BC5BD, etc.)
```

## Spacing

### Margins
```
Component Gap:    20px
Section Gap:      30px
Row Gap:          20px
Card Padding:     25px
```

### Box Model
```
Stat Cards:       25px padding, 10px border-radius, 4px left border
Progress Bars:    6px height, 3px border-radius
Rank Badges:      30px × 30px circle, centered
```

## Animation

### Fade In
When dashboard loads, elements fade in smoothly:
```css
animation: fadeIn 0.3s ease-in;
```

### Hover Transform
Cards elevate on hover:
```css
transform: translateY(-5px);
transition: all 0.3s ease;
```
