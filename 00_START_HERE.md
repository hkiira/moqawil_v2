## 🚀 Quick Start Command

To add a stat card to your page:

```php
<?= $this->element('dashboard/stat_card', [
    'title' => 'Total Orders',
    'value' => '2,456',
    'label' => 'Orders this month',
    'icon' => 'fa-shopping-cart',
    'type' => 'primary'
]) ?>
```

That's it! The component handles the rest.
